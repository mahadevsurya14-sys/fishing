About PhishGuard

PhishGuard is a cybersecurity web application built using HTML, CSS and JavaScript. It detects suspicious URLs, domains, IP addresses and uploaded files using heuristic analysis. This demo performs client-side analysis and simulates multiple security engines similar to enterprise threat intelligence platforms.
